## 📸 Flexbox Photo Gallery (HTML & CSS Project)

### 📌 Preview  
This project focuses on building a responsive photo gallery webpage using CSS Flexbox, designed to adapt to different screen sizes.

---

### 📖 About the Project  
In this project, I learned how Flexbox makes it easier to create flexible and responsive layouts. I used it to build a photo gallery that automatically adjusts images and layout based on the screen size, making the design more user-friendly across devices.

---

### 🚀 What I Learned  
- How **CSS Flexbox** works for layout design  
- Creating **responsive web layouts** that adapt to different screen sizes  
- Aligning and distributing items using Flexbox properties  
- Building a **photo gallery layout** using flexible containers  
- Improving mobile and desktop user experience with responsive design  

---

### 🛠️ Technologies Used  
- HTML5  
- CSS3 (Flexbox)  

---

### 🎯 Key Takeaway  
This project helped me understand how Flexbox simplifies responsive design and makes it much easier to build layouts that work across all screen sizes.
